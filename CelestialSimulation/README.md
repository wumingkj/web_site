## 一、设计意图

本软件旨在为天文学教学提供一个直观、交互式的模拟工具。通过实时3D可视化展示太阳系行星运动，并提供引力实验沙盒，帮助理解万有引力定律、开普勒定律及多体混沌现象。主要面向中学生及非专业天文爱好者，以实时模拟界面降低学习门槛。

## 二、程序功能特点

| 模块 | 功能 |
|------|------|
| **主菜单** | 星空粒子背景，键盘选择进入各场景 |
| **太阳系模拟** | 基于开普勒轨道方程直接计算 17 个天体位置（八大行星 + 主要卫星），**支持任意时间跳转**、时间滑块拖拽、正反向播放、焦点切换、轨迹显示、纹理贴图 |
| **引力实验沙盒** | 大质量中心 + 双星绕转混沌系统，可拖拽发射/右键删除粒子，自定义 1/r^1.5 引力律，**C 键打开创建面板**配置质量/半径/颜色/自转参数，支持暂停/继续、重置 |
| **命令行交互模式** | 纯文本模式，支持 `run`(进度条)/`step`/`status`/`focus`/`orbit`/`data`/`export` 等命令，方向键翻阅历史 |

- **太阳系场景**：Python 开普勒方程求解器，O(1) 任意时间跳转，完全可逆
- **引力沙盒**：基于 **REBOUND** N体模拟库（`ias15` 积分器），带软化核和自定义幂律引力，`enter()`/`exit()` 自动清理状态保证场景切换不卡顿
- 基于 **ModernGL + GLFW** 实现高性能 3D 渲染，支持 165+ FPS
- 17 种纹理贴图还原行星外观，星空背景图
- 弧形球相机（中键拖拽旋转、滚轮缩放、焦点硬锁定）
- OpenGL 3.3+ 自动降级到 CLI 文本模式
- CLI 模式支持键盘方向键上下翻阅命令历史（Windows 基于 `msvcrt` 原生实现）

## 三、核心算法

1. **太阳系：开普勒轨道求解器**  
   基于轨道根数（半长轴 a、偏心率 e、倾角 inc）直接计算天体位置：
   - 平近点角 M = 2π · t / 周期  
   - 牛顿迭代求解开普勒方程 M = E - e·sin(E)  
   - 计算轨道平面位置 → 旋转到参考系  
   - **O(1) 计算**，任意时间跳转无延迟，完全可逆（负时间）

2. **引力沙盒：N体数值积分**  
   使用 REBOUND 提供的 `ias15` 自适应高阶积分器，精确处理三体混沌和近距离交会。添加引力软化核（softening=0.3）防止近距离力发散。

3. **自定义引力律（沙盒可选）**  
   在牛顿 1/r² 基础上通过 `additional_forces` 接口叠加差值修正，实现 F ∝ 1/r^1.5 有效幂律：近距离力弱于牛顿（不易弹射），远距离强于牛顿（影响范围更大）。

4. **实时渲染优化**  
   - 使用 OpenGL 顶点数组对象（VAO）缓存球体网格，减少 CPU-GPU 传输
   - 文字纹理 GL 缓存（`FontAtlas._tex_cache`），避免每帧创建/销毁纹理
   - 轨迹点限制 150 个，自动移除旧点
   - 最小视觉半径保底（0.0004 AU），小卫星不至于消失

5. **列主序矩阵兼容**  
   `pyrr` 库返回列主序矩阵，全部统一转为行主序计算后 `.T.tobytes()` 送入 OpenGL，确保 MVP 变换正确。

## 五、运行说明

### 环境要求
- 操作系统：Windows 10（64位）
- Python 3.8+
- 支持 OpenGL 3.3+ 的显卡（可选，兼容自动降级到 CLI 模式）

### 安装与运行步骤

```powershell
# 创建虚拟环境
python -m venv .venv
.venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 运行 GUI 模式
python main.py

# 或强制命令行模式
python main.py --cli
```

### GUI 操作说明

#### 主菜单
| 按键 | 功能 |
|------|------|
| `↑` / `↓` | 选择菜单项 |
| `Enter` / `空格` | 确认选择 |
| `1` / `2` / `3` | 快速跳转 |
| `ESC` | 退出程序 |

|------|------|
| `空格` | 暂停/继续 |
| `+` | 加速（过零自动反向） |

| `↑`/`↓`/`W`/`S` | 切换焦点天体 |
| **时间滑块** | **点击/拖拽任意跳转时间** |
| **`<` `>` 按钮** | **微调 ±0.5 年** |
| 滚轮 | 缩放视角 |
| 中键拖拽 | 旋转视角 |
| `ESC` | 返回菜单 |

| `C` | 打开/关闭创建星球面板 |

| `✓ 保存` | 保存配置并关闭面板 |

| `ESC` | 返回菜单 |

### CLI 命令行模式命令

| 命令 | 功能 |
|------|------|
| `run <年>` | 运行模拟 N 年（显示进度条） |
| `step` | 单步运行（约 1 天） |
| `status` | 显示所有天体当前位置 |
| `focus <n>` | 切换焦点天体 |
| `orbit` | 显示焦点天体的详细轨道状态 |
| `data` | 显示初始轨道根数 |
| `export` | 导出模拟数据到 CSV 文件 |
| `help` | 显示帮助 |

CLI 模式下支持 **方向键 `↑`/`↓` 翻阅命令历史**（Windows 基于 `msvcrt`，Unix 基于 `readline`）。

### 导出数据格式

`export` 命令导出的 CSV 文件包含：
- 模拟时间、时间步长、积分器等参数
- 所有天体的位置、速度
- 以 `#` 开头的注释行包含元数据

### 打包

```powershell
pyinstaller --onefile --console `
  --add-data "assets;assets" `
  --add-binary ".venv\Lib\site-packages\glfw\glfw3.dll;." `
  --add-binary ".venv\Lib\site-packages\librebound.cp38-win_amd64.pyd;." `
  --collect-all rebound `
  -i icon.ico main.py
```

## 六、引用的库和资源

1. **Python 库**
   - **REBOUND**：高性能 N 体模拟库（引力沙盒用）
   - **ModernGL**：跨平台 OpenGL 绑定库，用于 3D 渲染
   - **GLFW**：窗口管理库
   - **Numpy**：数值计算库
   - **Pillow**：图像处理库，加载纹理
   - **pyrr**：矩阵运算库（列主序约定）

2. **资源素材**
   - **NASA/JPL**：提供太阳系行星的轨道参数数据
   - **行星纹理图**：从公开资源获取
